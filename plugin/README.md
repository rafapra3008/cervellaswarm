### Comandi

- `/init-sncp` - Inizializza memoria esterna del progetto

### Sistema SNCP

Sistema Nervoso Centrale Progetti - Memoria esterna su disco.

```
.sncp/
├── idee/              # Idee, ricerche
├── memoria/
│   ├── decisioni/     # Decisioni (con PERCHE!)
│   ├── sessioni/      # Log sessioni
│   └── lezioni/       # Lezioni apprese
└── coscienza/         # Stato corrente
```

## Quick Start

```bash
# 1. Installa il plugin
/plugin install https://github.com/cervellaswarm/plugin

# 2. Inizializza SNCP nel tuo progetto
/init-sncp

# 3. Usa gli agenti!
# Gli agenti sono automaticamente disponibili nel Task tool
```
