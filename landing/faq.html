🌟 General

What is CervellaSwarm? +
CervellaSwarm is an AI-powered development team that coordinates 17 specialized agents to help you build software faster. Instead of one general AI assistant, you get a full team: frontend, backend, tester, reviewer, security, and more - all working together with shared memory.
How is it different from Cursor or Copilot? +
  • Multi-agent: 17 specialized agents vs 1-4 generic ones
  • Memory: SNCP persistent memory vs session-only context
  • Quality gates: 3 Guardians verify every output
  • IDE agnostic: Works with any editor, not locked to VSCode
  • Privacy: Your code never leaves your machine
Who is CervellaSwarm for? +
CervellaSwarm is built for professional developers working on complex projects. It's especially valuable for:
  • • Solo developers managing full-stack projects
  • • Small teams (2-10 devs) who need to move fast
  • • Privacy-first environments (healthcare, finance)
  • • Anyone tired of re-explaining context to AI

⚙️ Technical

What languages and frameworks does it support? +
CervellaSwarm supports any language or framework that Claude supports - which is virtually all of them. Our agents have specialized knowledge in:
  • • Frontend: React, Vue, Svelte, Next.js, Tailwind
  • • Backend: Python, Node.js, FastAPI, Express, Django
  • • Mobile: React Native, Flutter, Swift, Kotlin
  • • DevOps: Docker, Kubernetes, AWS, GCP
  • • And many more...
Does it work offline? +
CervellaSwarm requires an internet connection to communicate with the Claude API. However, all your code and SNCP memory stays on your local machine - nothing is stored on our servers.
Can I use my preferred IDE/editor? +
Yes! CervellaSwarm is IDE-agnostic by design. It runs as a CLI tool alongside any editor you prefer - VS Code, Vim, Neovim, Emacs, JetBrains IDEs, Sublime, or any other. This is one of our key differentiators.
What is SNCP? +
SNCP (Sistema Nervoso Centrale del Progetto) is our persistent memory system. It's a folder structure (.sncp/) in your project that stores:
  • • Decisions: Why you chose certain technologies
  • • Ideas: Feature plans and brainstorms
  • • Context: Current session state
All agents share this memory, so you never have to re-explain your project context.

🔒 Privacy & Security

Is my code safe? +
Yes. Your code is processed through the Claude API with Anthropic's enterprise-grade security. We don't store any of your code on our servers. The SNCP memory lives entirely on your local machine.
Is my code used for AI training? +
No. We use the Claude API which does not train on user data. Your code is processed for your task only and not stored or used for model training. This is perfect for proprietary codebases.
Can I use it for enterprise/regulated industries? +
Yes! CervellaSwarm is designed with privacy-first industries in mind. The CLI runs locally, SNCP stays on your machine, and we don't log or store your code. Many users work in healthcare, finance, and other regulated sectors.
