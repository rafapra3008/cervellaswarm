## 2. PSICOLOGIA: PERCHÉ MENSILE PUÒ ESSERE PIÙ ATTRAENTE

### 2.1 Barrier to Entry BASSO

```
LICENZA PERPETUA 8.000€:
"Devo trovare 8.000 EUR ORA"
→ Decisione pesante, cash-flow impact immediato

CANONE MENSILE 350€:
"350€/mese ce li abbiamo nel budget operativo"
→ Decisione leggera, nessun impact cash-flow
```

**VANTAGGIO:** Cliente non deve "trovare soldi", li ha già nel budget corrente.

---

### 2.2 Framing Come "Servizio", Non "Prodotto"

```
LICENZA = PRODOTTO
"Compriamo un software"
→ Percezione: investimento una tantum, poi "ce lo gestiamo noi"

CANONE = SERVIZIO
"Qualcuno ci gestisce la contabilità digitale"
→ Percezione: outsourcing, zero pensieri, tutto incluso
```

**VANTAGGIO:** Cliente percepisce come "esternalizzazione problema" non "acquisto tool".

---

### 2.3 Risk Reversal (Reversibilità Rischio)

```
LICENZA PERPETUA:
"Se tra 2 anni non serve più, abbiamo speso 8k+"
→ Rischio percepito ALTO (sunk cost futuro)

CANONE MENSILE:
"Se tra 2 anni non serve, semplicemente smettiamo"
→ Rischio percepito BASSO (no commitment lungo termine)
```

**VANTAGGIO:** Cliente si sente "libero di uscire" (anche se probabilmente non lo farà mai).

---

### 2.4 Comparabilità con Altri Costi Ricorrenti

```
350€/mese è PARAGONABILE a:
- Commercialista mensile (300-800€/mese)
- Software gestionali (200-600€/mese)
- Servizi cloud business (100-500€/mese)

MESSAGGIO:
"Costa meno del vostro commercialista, ma vi fa risparmiare
10 ore/mese di lavoro!"
```

**VANTAGGIO:** Cliente ha già "budget mentale" per costi ricorrenti simili.

---

### 2.5 No "Buyer's Remorse"

```
LICENZA 8.000€:
Dopo firma: "Cavolo, abbiamo speso 8k!"
→ Buyer's remorse possibile

CANONE 350€:
Dopo firma: "Ok, 350€/mese, come Netflix aziendale!"
→ Zero buyer's remorse
```

**VANTAGGIO:** Cliente non ha "peso psicologico" post-decisione.

## 3. COSA INCLUDERE NEL CANONE

### 3.1 TUTTO INCLUSO (no extra)

Il canone mensile DEVE essere all-inclusive per funzionare psicologicamente:

**✅ Software & Licenza**
- Accesso completo sistema
- Tutti gli aggiornamenti
- Nuove feature (piccole)

**✅ Infrastruttura & Hosting**
- Server cloud
- Database
- Backup automatici
- Monitoraggio 24/7

**✅ Supporto Prioritario**
- Email/chat entro 24h
- Bug fix entro 48h
- Training remoto se serve

**✅ Manutenzione & Aggiornamenti**
- Sicurezza patches
- Miglioramenti UX
- Ottimizzazioni performance

**✅ Garanzie**
- Uptime 99%
- Privacy dati (GDPR compliant)
- Export dati sempre disponibile

---

### 3.2 COSA NON INCLUDE (Chiarire!)

Per evitare aspettative sbagliate, COMUNICARE cosa NON è incluso:

**❌ Feature Completamente Nuove**
- Esempio: Integrazione con gestionale hotel diverso
- Sviluppo custom maggiore
→ Queste si quotano separatamente

**❌ Training On-Site**
- Training remoto SÌ incluso
- Trasferta fisica NO (salvo accordo)

**❌ Consulenza Fiscale**
- Software gestisce contabilità
- Interpretazione fiscale rimane a commercialista

**❌ Supporto 24/7 Immediato**
- Risposta entro 24h lavorative
- Emergenze vere: best effort

## 4. GESTIONE OBIEZIONE "MA COSÌ PAGO PER SEMPRE!"

Questa è l'obiezione PRINCIPALE al modello subscription. Risposte efficaci:

### Risposta A: Reframing Costo

```
CLIENTE: "Ma così pago per sempre!"

RISPOSTA:
"Giusto. Ma vediamo i numeri:

Con licenza perpetua: 8.000€ oggi + 200€/mese manutenzione
→ Anno 1: 10.400€, Anno 2: 2.400€, Anno 3: 2.400€
→ TOTALE 3 anni: 15.200€

Con canone mensile: 350€/mese
→ TOTALE 3 anni: 12.600€

Risparmiate 2.600€ nei primi 3 anni, e NON dovete trovare
8.000€ oggi!"
```

**PRINCIPIO:** Mostra che "per sempre" costa MENO nel medio termine (3-5 anni).

---

### Risposta B: Focus Servizio

```
CLIENTE: "Ma così pago per sempre!"

RISPOSTA:
"Esatto! Come pagate per sempre elettricità, internet,
commercialista. Perché? Perché sono SERVIZI che usate
ogni mese.

Questo non è un software da installare e dimenticare.
È un servizio vivo:
- Server che gira 24/7
- Backup giornalieri
- Supporto quando serve
- Aggiornamenti continui

Finché lo usate, lo supportiamo. Il giorno che non serve,
smettete. Zero vincoli, zero penali."
```

**PRINCIPIO:** Sposta percezione da "prodotto" a "servizio continuativo".

---

### Risposta C: Protezione Investimento

```
CLIENTE: "Ma così pago per sempre!"

RISPOSTA:
"Capisco la preoccupazione. Ma pensiamoci:

Se comprate licenza 8.000€ e tra 2 anni Ericsoft cambia
formato PDF, o serve integrazione nuova... dovete pagare
sviluppo EXTRA. Sunk cost perso.

Con canone mensile, evoluzioni INCLUSE. Il sistema cresce
con le vostre esigenze. Non pagate per qualcosa che diventa
obsoleto."
```

**PRINCIPIO:** Canone = protezione da obsolescenza.

---

### Risposta D: Exit Strategy Chiara

```
CLIENTE: "Ma così pago per sempre!"

RISPOSTA:
"Nessuno vi obbliga! Il contratto è mensile, no vincoli.

Se domani decidete:
1. Vi diamo export completo dati (CSV, Excel)
2. Vi diamo backup database
3. Vi diamo documentazione sistema
4. Se volete codice sorgente, disponibile (prezzo separato)

Siete LIBERI. Non è vendor lock-in."
```

**PRINCIPIO:** Trasparenza exit = sicurezza psicologica.
