## SETUP PER REGISTRAZIONE

### Terminale
- Font: Monaco o SF Mono, size 18+
- Colori: Sfondo scuro, testo chiaro
- Finestra: Fullscreen o quasi

### Comandi da preparare
```bash
# Prepara questi prima di registrare:
spawn-workers --all
ls -la .sncp/
```

### Tools necessari
- **Recording:** QuickTime (Cmd+Shift+5) o OBS
- **Editing:** iMovie (gratis) o CapCut
- **Export:** 1080p, 30fps
