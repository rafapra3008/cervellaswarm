# Changelog

All notable changes to `cervellaswarm-agent-templates` will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-02-18

### Added

- 4 template agents: coordinator, quality-gate, architect, worker
- Generic `_shared_dna.md` template for shared team behavior
- `team.yaml` composition format for declarative team definitions
- 3 example teams: minimal (3 agents), standard (7), full (17)
- CLI tool `cervella-agent` with `init`, `init-team`, `list`, and `validate` commands
- Frontmatter YAML reference documentation
- Comprehensive test suite
