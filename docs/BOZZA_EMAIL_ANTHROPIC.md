Hi Anthropic team,

I'm Rafa, an independent developer who has built CervellaSwarm - a multi-agent
orchestrator specifically designed for Claude Code and Claude Desktop via MCP.

THE ARCHITECTURE:
- 16 specialized AI agents (not one generic assistant)
- 12 Workers (Sonnet) for execution: frontend, backend, testing, docs, security...
- 3 Guardians (Opus) for QA validation - they check the workers' output
- 1 orchestrator that coordinates everything
- Native MCP server support (npm install -g @cervellaswarm/mcp-server)

THE DIFFERENCE:
The unique aspect is that our AI team "checks its own work" -
dedicated QA agents review every output before it merges.

ALREADY IN USE:
I've used CervellaSwarm to build two production SaaS apps over the past months.
