# Studio Mercato No-Code/Low-Code per Non-Programmatori

**Data:** 6 Gennaio 2026
**Autore:** cervella-scienziata
**Tipo:** Ricerca Strategica/Business

---

## Executive Summary

Il mercato no-code/low-code vale $24.8 miliardi (2024) e crescera a oltre $65 miliardi entro 2027 (CAGR 38.2%). CervellaSwarm puo posizionarsi come "AI Building Platform" per non-tecnici, ma serve una UX RADICALMENTE diversa.

**RACCOMANDAZIONE:** Dual-track strategico. Non abbandonare i developer, ma esplorare il mercato non-tecnici con un prodotto derivato ("CervellaSwarm Studio").

**RISCHIO PRINCIPALE:** UX troppo complessa per non-programmatori. Servono investimenti significativi in design e onboarding.

## 4. AI + No-Code: Lo Stato dell'Arte

### 4.1 Come Viene Usata AI Oggi

| Uso AI | Piattaforma | Maturita |
|--------|-------------|----------|
| **Text generation** | Tutti | Alta |
| **Code suggestion** | FlutterFlow, Dify | Media |
| **UI generation** | v0, Bolt.new | Alta |
| **App generation** | Bolt.new, Create.xyz | Emergente |
| **AI Agents** | Softr, Dify | Emergente |
| **Multi-agent** | NESSUNO | Non esiste |

### 4.2 Cosa MANCA nel Mercato

| Gap | Descrizione | Opportunita |
|-----|-------------|-------------|
| **Multi-agent no-code** | Nessuno offre 16 agenti coordinati per non-tecnici | ALTA |
| **MAPPA visuale** | Nessuno mostra cosa sta facendo l'AI | ALTA |
| **Trasparenza processo** | Le AI "black box" spaventano | ALTA |
| **Orchestrazione visibile** | Chi fa cosa? Quando? | ALTA |
| **Controllo granulare** | Non "tutto o niente" | MEDIA |

### 4.3 Chi sta Combinando AI + No-Code

| Player | Approccio | Limiti |
|--------|-----------|--------|
| **Bolt.new** | Chat → App completa | Single-agent, black box |
| **v0** | Prompt → UI components | Solo frontend |
| **Create.xyz** | GPT-5 → Products | No trasparenza |
| **Softr AI** | AI assistant + Agents | Limitato a task semplici |
| **Dify** | LLM orchestration | Troppo tecnico |
