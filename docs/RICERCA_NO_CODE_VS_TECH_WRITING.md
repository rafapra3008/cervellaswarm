## EXECUTIVE SUMMARY

**TL;DR:** Technical Writing ha barriere più basse e timeline più predicibili. No-Code ha potenziale più alto ma richiede più hustle iniziale.

**Raccomandazione:** **Technical Writing** come punto di partenza (mesi 1-3), con possibile pivot/espansione verso No-Code Automation dopo primi €1500/mese stabili.

**Probabilità successo €1500/mese entro 3 mesi:**
- Technical Writing: **65-70%**
- No-Code Automation: **45-55%**

- Mese 4+: Retainer + progetti = €1,500+ stabile

**Fattori Critici:**
- **Hustle quotidiano** (cold outreach, networking)
- **Specializzazione** (es: automation per e-commerce, SaaS, etc.)
- **Case studies** documentati bene
