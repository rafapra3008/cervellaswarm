# Test Orchestrazione - CervellaSwarm

Questa cartella contiene test per validare l'orchestrazione dello sciame.

## Struttura

```
test-orchestrazione/
  api/          # Endpoint API di test
  components/   # Componenti frontend di test
  tests/        # Test automatici
```

## Come Usare

1. Esegui i test con pytest
2. Verifica che tutti passino
3. Controlla i log per dettagli

## Installation

```bash
cd test-orchestrazione
pip install -r requirements.txt
pytest
```
